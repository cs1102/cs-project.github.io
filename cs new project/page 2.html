Pinyin input method

The Pinyin input method is one of the most widely used Chinese input methods. Pinyin is a Romanization system for Mandarin similar to the Latin alphabet. Different companies design different versions of the Pinyin input method, for example, Sogou Pinyin and Google Pinyin. Still, the rationale behind it is the same, the user simply enters the pinyin of a Chinese character and selects the word from the recommendation.


Learn more about Hanyu Pinyin here

Advantages Disadvantages
  1. Pinyin is easy to learn for people with adequate Mandarin knowledge.
  2. The typing speed is fast once familiar since it uses the same QWERTY keyboard setup as most people.
  3. The method contains prediction and correction functions that increase typing speed.
  4. It provides a large word base.
  1. The method is complicated for the non-Mandarin speaker or people unfamiliar with pinyin.
  2. There are multiple versions of the pinyin input method.

Our experience:

Reymond: I prefer typing in double pinyin. You can type fast using consonant keys. Although it requires a lot of practice to get familiar with, it is much faster than typing the entire Pinyin of the word out.

Winkie: Since there are four tones and many homophones in Chinese, there is a high selection rate when using Pinyin input. Other than that, the method is convenient and easy to adapt.

Terrence: It is similar to typing in English. I get used to it very soon! Sometimes you might forget how to write a word, but using Pinyin input will never have this problem.

Kevin: I am not good at Mandarin, so I need to occasionally search for the pinyin of different words. The accuracy of the pinyin input method can also be affected by my pronunciation and spelling habits, lowering my typing efficiency.
