Stroke count method

The stroke count method is a relatively easy input method using the number pad to input (1-5), widely used in old-model mobile phones.


Each number represents one type of stroke, namely:

  1. (一), A horizontal stroke from left to right
  2. (丨), A vertical stroke from top to bottom
  3. (丿), A diagonal long stroke downward from right to left
  4. (丶), Dot: A very short dash stroke downward from left to right
  5. (フ), Zag: A horizontal stroke from left to right , ending with a downwards hook to the left

The user only needs to hit the keys that correspond to a character's strokes to enter it, then they can choose from a list of characters that match that character. As more digits of the code are entered, a narrower range of options are presented for selection.


Advantages Disadvantages
  1. It is easy to learn and use, as it requires no knowledge other than the Chinese word itself.
  2. It is pre-installed in some models of mobile phones, which is convenient to use.
  1. The stroke order is essential. The wrong order will result in the wrong word.
  2. It is slow compared to other input methods, especially for a character with many strokes
  3. The number of words it can recognize is limited.

Our experience:

Reymond: It is time-consuming to enter every stroke of a character, and there are often many suggestions to choose from. It is only suitable for writing a short message.

Winkie: Anyone with a basic Chinese stroke concept can learn it with minimum effort. My grandfather used to use it on his old phone.

Terrence: I seldom use it once I get familiar with other input methods. Its only advantage is that you don’t need to remember too many keys.

Kevin: It is the first Chinese input method I used besides handwriting. And it is straightforward to learn! It is handy for looking up unfamiliar characters, especially when I am unsure how to pronounce them.
