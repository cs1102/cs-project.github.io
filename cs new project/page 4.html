Cangjie input method


Chu Bong-Foo, a Chinese computer engineer who lived and worked in Taiwan between 1972 and 1978, created the Cangjie input method, which enables Chinese characters to be entered into a computer using a regular QWERTY keyboard.

A method of using a regular computer keyboard to type Chinese characters into a computer. It summed up 24 "Cangjie alphabets" and 76 "auxiliary fonts" derived from these basic letters according to the glyph structure and composition principles of Chinese characters.


Advantages Disadvantages
  1. The Chinese input method with the lowest word selection rate.
  2. Constantly updating and maintaining.
  3. It provides a large word base.
  1. You must know how to write that word in order to type it out.
  2. It takes time to learn the split word principle , and it is a bit complicated and difficult to learn.
